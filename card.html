دورات تكنولوجيا المعلومات

اختر الدورة التي تريد تعلمها من الكروت أدناه وابدأ رحلتك في عالم تطوير الويب

دورة HTML

تعلم لغة هيكلة المحتوى الأساسية لبناء مواقع الويب. ستتعلم كيفية إنشاء هيكل الصفحات، إضافة النصوص، الصور، الروابط والنماذج.

دورة CSS

أتقن فن تنسيق وتصميم صفحات الويب. ستتعلم كيفية جعل مواقعك جذابة باستخدام الألوان، الخطوط، التخطيطات والتأثيرات البصرية.

دورة JavaScript

أضف التفاعلية إلى مواقعك الويب. ستتعلم كيفية معالجة الأحداث، تعديل المحتوى ديناميكيًا، والتفاعل مع المستخدمين.

دورة HTML كاملة

لغة ترميز النص التشعبي (HTML) هي العمود الفقري لأي موقع ويب

ما هي HTML؟

HTML (لغة ترميز النص التشعبي) هي اللغة القياسية لإنشاء صفحات الويب. تصف HTML بنية صفحة الويب وتتكون من سلسلة من العناصر التي تخبر المتصفح كيف يعرض المحتوى.

ما سوف تتعلمه في هذه الدورة:

  • أساسيات HTML والعناصر الأساسية
  • إنشاء هيكل صفحة الويب
  • إضافة النصوص والعناوين والفقرات
  • عمل الروابط والصور
  • إنشاء الجداول والقوائم
  • بناء النماذج وعناصر الإدخال
  • أفضل الممارسات في كتابة HTML
  • HTML5 والعناصر الدلالية الجديدة

أمثلة على عناصر HTML:

العناصر هي اللبنات الأساسية لصفحات HTML. كل عنصر يتكون من وسم فتح ووسم إغلاق والمحتوى بينهما.

دورة CSS كاملة

أوراق الأنماط المتتالية (CSS) تستخدم لتنسيق وتصميم صفحات الويب

ما هي CSS؟

CSS (أوراق الأنماط المتتالية) هي اللغة المستخدمة لوصف مظهر وتنسيق المستندات المكتوبة بلغة HTML. تتحكم CSS في الألوان، الخطوط، المسافات، التخطيط العام والعديد من الجوانب البصرية الأخرى لصفحة الويب.

ما سوف تتعلمه في هذه الدورة:

  • مقدمة إلى CSS وطرق دمجها مع HTML
  • محددات CSS والأولويات
  • تنسيق النصوص والخطوط
  • الألوان والخلفيات
  • النموذج الصندوقي (Box Model)
  • التخطيطات والمواقع (Flexbox, Grid)
  • التصميم المتجاوب لجميع الشاشات
  • التحويلات والتحولات والرسوم المتحركة

لماذا CSS مهمة؟

بدون CSS، ستكون جميع صفحات الويب مجرد نصوص سوداء على خلفية بيضاء مع روابط زرقاء. تضيف CSS الجمال والتجربة البصرية التي تجعل مواقع الويب جذابة وسهلة الاستخدام.

دورة JavaScript كاملة

جافا سكريبت (JavaScript) تضيف التفاعلية والديناميكية إلى صفحات الويب

ما هي JavaScript؟

JavaScript هي لغة برمجة عالية المستوى تستخدم primarily لجعل صفحات الويب تفاعلية. تتيح JavaScript إنشاء محتوى ديناميكي، تحكم في الوسائط المتعددة، تحريك الصور، وتفاعل مع إجراءات المستخدم.

ما سوف تتعلمه في هذه الدورة:

  • أساسيات JavaScript والمتغيرات
  • أنواع البيانات والعوامل
  • الشروط والحلقات التكرارية
  • الدوال ووجهات النظر
  • كائنات JavaScript والمصفوفات
  • التعامل مع نموذج HTML (DOM)
  • أحداث JavaScript
  • مقدمة في jQuery وAJAX

أهمية JavaScript:

اليوم، JavaScript لا تقتصر على المتصفح فقط. يمكن استخدامها في تطوير تطبيقات الخوادم (Node.js)، تطبيقات الهواتف (React Native, Ionic)، وحتى في برامج سطح المكتب (Electron).
